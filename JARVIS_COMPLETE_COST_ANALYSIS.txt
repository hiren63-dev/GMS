COST OPTIMIZATION TIPS
================================================================================

1. USE GEMINI (saves 90% vs ChatGPT)
   - Switch to Google Gemini 2.0 Flash
   - Cost: ₹600-1,200/month for normal use

2. CACHE CONVERSATIONS
   - Current system caches last 30 messages
   - Avoids re-processing old context
   - Saves ~30% API cost

3. BATCH SCREENSHOTS
   - Take one screenshot and ask multiple questions
   - Don't take screenshot for each query

4. LOCAL VOICE ONLY
   - Using Whisper + Kokoro (both free)
   - Saves ₹3,000+/month vs cloud STT/TTS

5. SET MONTHLY BUDGET CAP
   - Jarvis has built-in budget limit
   - Currently set to $10/month (₹825)
   - Edit config.py to increase if needed

6. CLEAN UP OLD SESSIONS
   - Delete unused sessions every 30 days
   - Reduces database size → Supabase costs

7. USE TEXT ONLY WHEN POSSIBLE
   - Don't upload files/documents unless necessary
   - File processing costs more tokens

8. PRE-PROCESS LARGE DOCUMENTS
   - Split 100-page docs into 5-10 page chunks
   - Cheaper than processing whole doc at once
