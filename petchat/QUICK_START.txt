Now follow these 4 steps to get your app running:

================================================================================
STEP 1: FIREBASE SETUP (15 minutes)
================================================================================

1. Go to firebase.google.com
2. Sign in with Google account
3. Click "Create a project"
   - Name: petchat
   - Uncheck Google Analytics
   - Create
4. Enable Firestore Database
   - Build → Firestore Database → Create
   - Location: asia-south1 (for India) or closest to you
   - Test mode
5. Enable Authentication
   - Build → Authentication → Email/Password → Enable
6. GET FIREBASE CONFIG
   - Click gear icon → Project Settings
   - Scroll to "Your apps" → Web app
   - Copy the 6 values (apiKey, authDomain, projectId, etc)

================================================================================
STEP 2: CREATE .env.local FILE
================================================================================

In the petchat folder, create a NEW FILE called .env.local

Copy and paste your 6 Firebase values:

VITE_FIREBASE_API_KEY=AIzaSy...
VITE_FIREBASE_AUTH_DOMAIN=petchat-xxxxx.firebaseapp.com
VITE_FIREBASE_PROJECT_ID=petchat-xxxxx
VITE_FIREBASE_STORAGE_BUCKET=petchat-xxxxx.appspot.com
VITE_FIREBASE_MESSAGING_SENDER_ID=1234567890
VITE_FIREBASE_APP_ID=1:1234567890:web:abcdef123456

(Replace the xxxxx values with your actual Firebase values)

================================================================================
STEP 3: RUN LOCALLY
================================================================================

Open PowerShell in the petchat folder and run:

npm install
npm run dev

Open http://localhost:3000 in your browser

TEST:
- Click 📊 (admin button)
- Login with email/password (first create account in Firebase Console)
- Add employees
- Back to chat, select name, click mascot, send messages!

================================================================================
STEP 4: DEPLOY TO VERCEL
================================================================================

1. Push to GitHub
   - Create GitHub account & new repo called "petchat"
   - git init
   - git add .
   - git commit -m "Initial"
   - git push

2. Deploy on Vercel
   - Go to vercel.com
   - Sign up with GitHub
   - Import your "petchat" repo
   - Add environment variables (same 6 Firebase values from .env.local)
   - Deploy!

3. Share the URL with your team!
